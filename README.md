# create-cursor-workspace

A CLI tool to quickly set up a new Cursor workspace.

## Usage

```bash
npx create-cursor-workspace <project-name>
```

## Development

Clone the repository and install dependencies:

```bash
git clone <your-repo-url>
cd create-cursor-workspace
npm install
```

To run locally:

```bash
npm start <project-name>
# or
node index.js <project-name>
```
